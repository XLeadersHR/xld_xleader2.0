⚙ Cấu hình Microsoft Graph API

Nhập Client ID và Tenant ID từ Azure AD App Registration để kích hoạt chức năng "Gửi tự động". Cấu hình lưu cục bộ trong trình duyệt.

Lấy từ Azure Portal → App registrations → Overview
Tenant ID của tổ chức XLeaders, hoặc dùng common để chấp nhận mọi tài khoản Microsoft
Copy giá trị này và thêm vào Azure App Registration → Authentication → Platform: Single-page application
📘 Hướng dẫn setup Azure (~5 phút):
  1. Vào portal.azure.com → App registrations → New registration
  2. Đặt tên app, chọn Accounts in this organizational directory (single tenant)
  3. Sau khi tạo, copy Application (client) ID và Directory (tenant) ID → dán vào trên
  4. Vào Authentication → Add a platform → Single-page application → dán Redirect URI ở trên
  5. Vào API permissions → Add → Microsoft Graph → Delegated → tick Mail.Send + User.Read
  6. (Khuyến nghị) Bấm Grant admin consent cho cả tổ chức

🔥 Bức tường yêu thương dùng chung

Cấu hình Firebase Realtime Database để tất cả thành viên XTeam thấy chung bức tường (real-time sync). Để trống nếu chỉ muốn lưu cục bộ.

Vào Firebase Console → Project Settings → "Your apps" → Web app → copy nguyên đoạn firebaseConfig = {...}
